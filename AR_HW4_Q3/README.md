# AR Game - Unity Project

Auther: Jack Xie

## Description

A Unity-based AR game where players collect cubes, activate power-ups, and navigate through obstacles to reach the goal. The game features 3D movement, collectible items, power-up mechanics, and interactive walls that can only be broken when the power-up is active.


## Files Included

### Scripts
- `PlayerController.cs` - Main player logic, movement, and game mechanics
- `CameraController.cs` - Camera follow system for third-person view
- `Rotate.cs` - Object rotation script for animated elements

### Assets
- `Materials/` - Game materials and textures
- `Prefabs/` - Reusable game objects (the walls)
- `Scenes/` - Game scenes and objects


### Project Files
- `.gitignore` - Git ignore file for Unity projects


## How to Run

1. Open the project in Unity 2022.3 LTS or later
2. Load the SampleScene
3. Press Play to start the game
4. Use WASD to move, Space to jump
5. Collect all cubes to spawn a power-up capsule
6. Touch the capsule to gain enhanced abilities
7. Break walls and reach the goal zone to win
